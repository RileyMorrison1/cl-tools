# cl_to-do_list
## Description
cl_to-do_list is a simple command line to-do list. You can add and remove tasks from the list. The contents of the to-do list are stored inside of a text file.

## Operators
- 'n' - New task
- 'd' - Delete task
- 'r' - Read tasks
- 'c' - Clear tasks
- 'h' - Get help
- 'q' - Quit program
